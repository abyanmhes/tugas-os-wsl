# Tugas OS WSL — Todo CLI
Aplikasi task manager berbasis CLI menggunakan Python.
## Cara Menjalankan
```bash
python3 main.py add "nama task"
python3 main.py list
python3 main.py done 1
python3 main.py delete 1
```
## Environment
- WSL2 (Ubuntu)
- Python 3.x
- VS Code + Remote WSL
